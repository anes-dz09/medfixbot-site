📜 MedfixBot Privacy Policy

Last Updated: 29/07/2025

🔍 Information We Collect

MedfixBot only processes and temporarily stores:

  • Discord User IDs – for giveaway entrants, winners, command authors.
  • Guild IDs & Channel IDs – to configure and send messages.
  • Message IDs – to track and manage giveaways.

We do NOT collect:

  • Message content
  • DMs
  • Personal data
  • IP addresses

🎯 How We Use the Data

We only use the data to:

  • Run giveaways (/gstart, /gend, /greroll)
  • Store ongoing giveaway settings
  • Announce winners

⏳ Data Retention

Data is kept only during the giveaway's lifetime and cleared after.

🔐 Security & Sharing

  • No sharing with third parties
  • All communication is over HTTPS via the official Discord API

⚙️ Your Control

  • Admins can remove the bot anytime
  • Users can opt-out by not joining giveaways

📢 Updates

We’ll announce any changes to this policy via support server or bot description.

By using MedfixBot, you agree to this policy.
